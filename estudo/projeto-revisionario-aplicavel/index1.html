Formatações de texto

Negrito e Itálico semântico

Use a tag <strong> para fazer o Negrito Semântico. Ex: Negrito.

Use a tag <em> para usar Itálico em forma semântica. Ex: Itálico.

OBS: Para representar "<>" de forma semântica utilize <conteúdo>.

Outras formatações

Para marcar algo use a tag <mark>. Ex: Marcação

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, para isso utilize a tag <del>. Ex: Já fui cruzerense.

Podemos marcar um texto como inserido, utilizando a tag <ins>. Para mostrar que o texto foi inserido depois da primeira postagem de algo. Ex: FAKE NEWS.

Podemos marcar um texto como sobrescrito utilizando a tag <sup>. Ex: 42.

Podemos marcar um texto como subescrito utilizando a tag <sub>. Ex: H2O

Podemos usar a tag <code> para sinalizar que algo é um código da programação. Ex: <strong>.

A tag <pre> serve para colocar códigos do jeitinho que forma escritos. Ex:

Para fazermos uma citação simples de algo utilizamos a tag <q>. Ex: Como diria um pai de um amigo O computador é um burro muito rápido.

Para fazermos uma citação completa (que é aquele citação que é mais completa mais cheia de texto) de algo utilizamos a tag <blockquote> (Essa tag possui o parâmetro cite). Ex:
