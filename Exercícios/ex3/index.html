Negrito e Itálico semântico

Negrito em forma semântica: Negrito.
Use a tag <strong>

Você representa "<>" de forma semântica utilizando: "<" e ">"

Itálico em forma semântica: Itálico.
Use a tag <em>.

Outras formatações

Para marcar um texto utilize a tag <mark>.

Ex: Texto marcado


A tag <big> serve para deixar um texto maior, só que não é semântico o correto nesse caso é utilizar um estilo de CSS para aumentar o tamanho da letra.

Texto pequeno, use a tag <small>. Geralmente o sentido/significado semântico disso é para escrever aquelas letrinhas minúsculas de contrato e imagino que para atribuir autoria de algo.

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Utilizando a tag <del>.

Ex: Campeão Mundial de 1951 -> Palmeiras

Podemos marcar um texto como inserido, utilizando a tag <ins>. Pra mostrar que o texto foi inserido depois da primeira postagem de algo.

Ex: FAKE NEWS -> "Campeão Mundial de 1951 -> Palmeiras".
