Shorthand é um propriedade que condesa muitas declarações em uma declaração. O shorthand para font em CSS é font, a ordem dentro dela é: font-style; font-weight; font-size; font-family. Ex:


        h1 {
            font: italic bolder 3em 'Work Sans', sans-serif;
           }

Se você não quiser usar todas as declarações do shorthand font, tudo bem, ex:

            
        h2 {
            font: bolder 3em 'Work Sans', sans-serif;
        }
